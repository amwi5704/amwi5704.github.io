Challenge 5: Accessibility Concerns

A massive issue with the design of this website is the use of tables in challenge 2 and challenge 4. These elelments cannot be read properly by screen readers, so for someone with vision issues, they will end up being unintelegible. For the chess board, inserting an image of a chess board with alt text describing the position of the pieces would help, however if the aim is for the chess board to be functional within the website, much more thought would have to go into its design. Challenge 4 would encounter some accessability problems with the sub-links, however the rest of the page is fairly accessable. To solve this problem, investigating how google handles accessability would be an obvious first step.
